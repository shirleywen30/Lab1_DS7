# LABORATORIO 1 DE PHP

Este repositorio contiene una serie de ejercicios desarrollados en PHP, enfocados en lógica básica, validación de datos, formularios y uso de HTML + CSS.

---

## PROBLEMAS

### Problema 1: Área y perímetro de un círculo
Se solicita al usuario el radio de un círculo y se calcula:
- Área (π * r²)
- Perímetro (2 * π * r)

---

### Problema 2: Conversión de pulgadas a centímetros
Convierte un valor ingresado en pulgadas a centímetros.
- Fórmula usada: 1 pulgada = 2.54 cm  

---

### Problema 3: Información de PHP
Muestra información detallada sobre la configuración actual de PHP usando la función phpinfo().

---

### Problema 4: Uso de variables en PHP
Muestra un mensaje usando variables.

### Problema 5: Formulario de datos con validación
Verifica si el usuario es mayor de edad y puede votar o si no lo es.

Formulario solicita:
- Nombre
- Edad

Incluye:
- Trimming (elimina espacios)
- Sanitización (evita código malicioso)
- Normalización (formato del nombre)

---

### Problema 6: Calculadora web
Calculadora web con interfaz tipo aplicación.

Permite realizar:
- Suma (+)
- Resta (-)
- Multiplicación (*)

---
